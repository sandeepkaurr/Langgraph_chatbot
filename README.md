# LangGraph Chatbot with Streamlit

This is a simple chatbot built using **LangGraph**, **LangChain Groq integration**, and **Streamlit** for the frontend.  
It supports conversation memory and is powered by the LLaMA 3.1 model via **Groq API**.

## Features

- Conversational chatbot using **LangGraph state graph**
- Persistent memory using **InMemorySaver**
- Frontend using **Streamlit chat interface**
- Supports multi-turn conversation
- Easily extendable for multiple threads or custom nodes
